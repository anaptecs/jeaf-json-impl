# JEAF JSON API README #

This repository contains the source code of JEAF JSON API. JEAF JSON contains a set of classes that simplify usage of JSON (Jackson 2). This projects contains public API classes that belong to JEAF JSON.

## Links ##
For further information please refer to:

* [JEAF JSON](https://anaptecs.atlassian.net/l/c/fBmaPk18)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.json/jeaf-json-api)
* [Javadoc](https://javadoc.io/doc/com.anaptecs.jeaf.json/jeaf-json-api)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
